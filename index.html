CSS Flexbox

I have chosen to only implement CSS-Flexbox as it enabled more inter dynamic element widths without being too constrained.

Since I have implemented a row wise element style, using flexbox for each row containing multiple elements in only one dimension made sense

If I wanted to have divs with multiple rows and columns inside I would prefer to use grid, as it offers complete control over multi-element positioning within divs.

HTML Canvas

Animation of canvas is based on redrawing individual frames, making it harder to keep exact time as in SVG. Had to experiment with where the objects ended up in a certain time due to difference in travel lengths in the animation.

In comparison with SVG, you have to clear the whole image and continously redraw all the figures/frames

Have to move a "cursor" to draw by drawing lines or arcs. Instead of SVG that has a better API with existing shapes like lines, circles or polygons where you just have to supply coordinates of the corners

By having complete control over every frame that is drawn, the animation can easily be paused by not requesting the next frame. This way the still image of where the animation left off is showed.

If the animations should move independently of each other in terms of time, canvas is a great choice.

Made the canvas interactive by changing color variables and then redrawing the frame to update the color, without jQuery (click to try)

SVG

Based on time, could set 10 seconds and the whole animation conformed to that time interval

Do not have to redraw the figures, just change the parameters of the figures

When the animation is stopped, the image is returned to the original image and not where the animation left off. To fix this the drawing should be manually redrawn and the coordinates of where the animation left off should be calculated by elapsed time.

If the animations are time sensitive, this is the best option

The SVG could be easily manipulated with jQuery to change colors of the different objects. Click the image to try!

jQuery

Used jQuery to dynamically change CSS by first selecting the item id and then manipulating the css

This way i could hide and show a popup upon buttonpresses

Used jQuery to load text from text files into paragraph section in documentation using jQuerys Ajax that reads the file and inputs the stream of text into the element with the given id (NB: This turned out not to work when outside the IDE! But can be seen in the textLoader.js file)

In order to change colors of the SVG image, the individual objects of the image were selected and the color attributes were changed.

Tested in Microsoft Edge, Chrome and Mozilla Firefox. Canvas, SVG and jQuery functionality works identical across browsers

CSS also displays correctly in all browsers
